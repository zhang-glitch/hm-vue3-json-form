# zlui

## project catalogue

- `components` 通用组件库

- `demos` schema 测试测试

- `lib` 项目代码

  - `fields` input 分类组件
  - `SchemaForm` 暴露的组件
  - `SchemaFormItem` 过度组件，主要是整合 fields 定义的组件
  - `types` 类型定义 （props, interface schema）

- `App` demo 测试封装组件
